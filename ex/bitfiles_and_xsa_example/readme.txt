This folder contains a .xsa file, 2 partial and 1 full .bit-file. 
The two .bit-files pulls either hd_gpio 7 or 8 high. 
XPAR_AXI_GPIO_1_BASEADDR pin 0 is constrained to hd_gpio_5 to use as debugger or benchmarking
use the .xsa file as platform in vitis, and load the 2 partial -bit-files onto a sd_card with a exfat partition.
